== Description ==

Plugin for detailed listing of your yachtino (yachtall, happycharter) offers.

== Installation ==

1. Upload `shiplisting.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
